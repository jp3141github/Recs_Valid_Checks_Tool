# Archive

This directory contains non-critical files that have been archived from the main codebase.

## Contents

### utilities/
Development and testing utilities:
- `generate_synthetic_data.py` - Generates realistic synthetic datasets for testing
- `create_config_template.py` - Creates Excel configuration templates

### docs/
Documentation files:
- `system_design.pdf` - System architecture and design documentation

### backups/
Backup and archive files:
- `recon_tool.zip` - Previous backup of the tool

## Usage

These files are not required for normal operation but can be useful for:
- Setting up test environments
- Generating sample data for demos
- Creating new configuration files
- Reference documentation
